A simple random string generator as a Flask app. Made as a Flask learning project.

To use locally:

1. Clone repo

        git clone https://github.com/alienemoji/flask-str-gen
        cd flask-str-gen
        
2. (Recommended) Create and activate a venv

        python3 -m venv env
        source env/bin/activate
        
3. Install requirements

        pip install -r requirements.txt
        
4. Run app

        export FLASK_APP=app.py
        flask run
        
5. Point browser to localhost:5000
